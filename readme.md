### Overview
This project is a community‑focused website designed to support families and individuals experiencing hardship. It provides access to essentials, such as clothing donations, food, furniture, and household electrical items. The website aims to offer a point of contact for those that may be in need. The CIC has a commitment to improving community wellbeing

## User Stories

### 1. As a community member  
I want to understand what Carolina’s Free Shop does, so that I know how it can help me.

- **Feature:** Home page opening message  
- **Acceptance Criteria:**  
  - The homepage clearly explains the purpose of the CIC  
  - Opening hours are visible  
  - Navigation links to Free Shop and Contact Us pages  
- **Evidence:** Homepage displays mission, opening hours, and clear navigation.
![Homepage displays mission, opening hours, and clear navigation.](assets/screenshots/home-page.png)
---

### 2. As a visitor  
I want to see what items are available in the Free Shop, so that I know whether it is worth visiting.

- **Feature:** Free Shop information page  
- **Acceptance Criteria:**  
  - Page lists typical items available  
  - Page lists items accepted and not accepted  
  - Page explains how the shop works  
- **Evidence:** Free Shop page explains what the shop is about.
![Free Shop page explains what the shop is about.](assets/screenshots/free-shop-page.png)

---

### 3. As a donor  
I want to know what items I can donate, so that I can give appropriate items.

- **Feature:** Donation guidance section  
- **Acceptance Criteria:**  
  - Clear list of acceptable items  
  - Clear list of items not accepted  
- **Evidence:** Free Shop page includes donation rules and contact link.
![Free Shop page includes donation rules and contact link.](assets/screenshots/types-of-donations.png)
---

### 4. As a visitor  
I want to know the location and opening hours, so that I can plan my visit.

- **Feature:** Location & Opening Hours  
- **Acceptance Criteria:**  
  - Address is displayed  
  - Opening hours are displayed  
  - Page is mobile‑friendly  
- **Evidence:** footer includes address and hours.
![footer includes address and hours.](assets/screenshots/footer.png)

### 5. As a volunteer  
I want to understand how I can help, so that I can decide whether to get involved.

- **Feature:** Volunteer Information page  
- **Acceptance Criteria:**  
  - Page explains volunteer roles

  - Page is easy to navigate

- **HTML5** – Provides the structure and semantic layout of the website.  
- **CSS3** – Handles styling, layout, and visual presentation.

## Testing
The website was tested throughout development to ensure functionality, responsiveness, accessibility, and performance. A combination of manual testing, Lighthouse audits, and W3C validation tools were used. Key findings and fixes are documented below.

**1. Manual Testing**

**Navbar collapsing too early on Chrome**

**Issue:**

 Navbar collapsed above 992px on Chrome but not on Edge.

**Cause:**
 
Horizontal overflow and a broken layout due to an extra closing. 

**Fix:**

Removed the extra closing tag.
Set carousel images to width: 100% and display: block.

Added overflow-x: hidden to html, body.
Changed Bootstrap class from navbar-expand-lg to navbar-expand-md for more stable behaviour.

 **Large image file sizes affecting performance**

**Issue:** Lighthouse flagged logo and images as too large. I reduced the images but decided againt logo as having previously tried the quality was reduced and logos were pixelated.

**Fix:** Converted all images from PNG/JPG to WEBP and reduced pixel dimensions using Affinity.

**Result:** Faster loading and improved Lighthouse performance scores.

**Empty space appearing on Chrome**

**Issue:** Large blank area below content.

**Cause:** Extra closing 
breaking the grid.

**Fix:** Removed the incorrect tag and retested.
